AI-powered sorting plus depth-aware bin monitoring powered by Luxonis OAK-D Pro.

Two Problems, One Eco-Island

Problem 1: User Mistakes

Public bins rely on users choosing correctly. One wrong item can contaminate an entire recycling stream.

Problem 2: Bin Operations

Wrong objects inside bins and early collection both create high costs: cleanup, rejected batches, and unnecessary pickup routes.

Image source: colorado.edu

Complementary Solutions

The two parts work together, which is why they deserve separate dashboard views.

Solution 1: Automatic Sorting

The system classifies the item at the deposit point and routes it to the correct trash category.

Solution 2: Verify & Monitor Bins

Depth tracking detects the wrong object position, notifies the operator dashboard, and estimates whether the bin or eco-island is actually full.

Eco-Island Workflow

1

Authenticate

User identifies themselves via card or app to begin the disposal session.

2

Deposit

User places the waste item into the designated central detection area.

3

Identify & Route

RGB-based inference identifies the waste category and routes the object to the right bin.

4

Depth Verify

After routing, depth analysis tracks where the object lands, checks for wrong-bin placement, and estimates fullness.

Why OAK-D Pro: Depth Is the Point

RGB for What It Is

On-device neural inference classifies the waste material at the moment of disposal.

Depth Estimation

Stereo depth tracks the object position inside the bin and estimates remaining capacity, turning the camera into an operations sensor.

Feature 2: Wrong Items & Collection Costs

Wrong-Object Detection

If an item ends up in the wrong bin, the system flags it instead of treating the bin as clean.

Position Tracking

Depth perception records where the object landed, so operators know where to retrieve it.

Fullness Estimation

The same depth map estimates whether the trash can or eco-island is actually full.

Smarter Pickup Timing

Operators avoid expensive collection trips when bins still have usable capacity.

Future Development: Anonymous Brand Insights

  • The high-resolution RGB sensor can identify selected product brands, not only material types.
  • This can create aggregate consumption statistics for a location or eco-island.
  • It is a separate analytics layer from user authentication.
  • Authentication is only for starting the disposal session, not for tracking brand behavior by person.

Architecture: Two Dashboard Views

Sorting Dashboard: item class, confidence, selected bin, and automatic routing state.
Bin Monitor Dashboard: wrong-object alerts, object coordinates, fullness, and pickup readiness.
RGB-D Input: aligned RGB frames and depth maps from the OAK-D Pro.
Backend Services: user session, classification events, spatial events, and notifications.
